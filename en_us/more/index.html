Kalestia translations for WhatsApp on iOS

You can download a translation file on your iPhone to show WhatsApp in Kalestia.

Download language file

Important notes

  • The process requires your iPhone to be jailbroken. If you don't know what a jailbreak is, there is a 99% chance you are not jailbroken and it won't work.
  • The translation file isn't by any means complete at this point in time which means that many strings will still show in English. Keep coming back to the site as the file will be updated frequently.
  • The file was tested and works on WhatsApp version 2.19.92 on iOS version 12.4. Other versions may work but have not been tested.
  • I am not responsible for any black screens, bootloops or dead phones. Editing system files can be dangerous so please be aware of what you are doing!

Instructions for setting up

  1. Again, please make sure your phone is jailbroken or else it won't work. There is no way you can do that without a jailbreak.
  2. If you haven't done so yet, download a file manager that can show system files (I recommend Filza).
  3. In your file explorer navigate to /var/containers/Bundle/Application/9CB5B1D1-2B9C-4D5D-9C40-E0394A869379/WhatsApp.app/, then into whichever language folder of which your system is set to. If your system is set to English, that folder would be en.lproj.
  4. Backup the original language file by renaming Localizable.strings to Localizable.strings.bak.
  5. Now move the downloaded language file to that same folder without renaming. It has to be called Localizable.strings.
  6. Kill WhatsApp from your app switcher and reopen it. You should be good to go!

Instructions for restoring

  1. In your file explorer navigate to /var/containers/Bundle/Application/9CB5B1D1-2B9C-4D5D-9C40-E0394A869379/WhatsApp.app/, then into whichever language folder of which your system is set to. If your system is set to English, that folder would be en.lproj.
  2. Delete the custom language file Localizable.strings.
  3. Rename Localizable.strings.bak back to Localizable.strings.
  4. Kill WhatsApp from your app switcher and reopen it. You should be good to go!
